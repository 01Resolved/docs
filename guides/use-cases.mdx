## 1. Weekly ecosystem snapshot

Get token price, treasury, NAV, mNAV, premium/discount, and runway for every tracked project in a single call.

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/global-dashboard/projects-launch"
```

Returns all 12+ ownership coins with `tokenUsdPrice`, `treasuryValue`, `netAssetValue`, `mNAV`, `premDisc`, and `monthsOfRunway` for each.

---

## 2. Monitor all active proposals

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/global-dashboard/decision-markets?limit=50"
```

Filter on `status` field for `live` proposals. Sort by `volume` to prioritize by significance.

---

## 3. Check if a live proposal is passing

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/proposal/FG93KeoUdUmBtTcxWhJZicc6ef824qccpFdesbw2RJjM/pass-edge"
```

Check `proposalTwapStatus`: `approve`, `reject`, or `pending`. Use `approveRejectTwapDifference` for the current margin.

## 4. Full treasury breakdown for a project

```bash
# Headline: total balance, NAV, runway
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/dao/treasury/overview?slug=metadao"

# Detail: every wallet and every token held
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/dao/treasury?slug=metadao"
```

## 5. Get the full text of a proposal

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/proposal/FG93KeoUdUmBtTcxWhJZicc6ef824qccpFdesbw2RJjM"
```

## 6. Analyze all trades on a decision market

```bash
# All trades, paginated
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/proposal/FG93Keo.../orders?limit=100"

# Trade conviction breakdown
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/proposal/analytics/FG93Keo.../outcome-strength"

# How conviction changed over time
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/proposal/analytics/FG93Keo.../conviction-chart"
```
