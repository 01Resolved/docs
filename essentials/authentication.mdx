## API Keys

All requests to the 01Resolved API require an API key passed in the request header.

| Header | Value |
|--------|-------|
| `x-api-key` | Your API key |

```bash
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.01resolved.com/v1/dao/treasury/overview?slug=metadao"
```

Requests without a valid API key return `401 Unauthorized`.

## Getting an API Key

The API is currently in **beta**.

  Message the 01Resolved team on Telegram with a brief description of what you are building.

Once approved you will receive an API key to include in all requests.

## Checking Your Key

Use the `/v1/api-key/current` endpoint to verify your key and check its configuration.

```bash
curl -H "x-api-key: YOUR_API_KEY" \
  "https://api.01resolved.com/v1/api-key/current"
```
