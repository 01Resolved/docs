## Quick start

<Steps>
  <Step title="Request access">
    Message the 01Resolved team on [Telegram](https://t.me/Official01Resolved) with a brief description of what you are building. You will receive an API key once approved.
  </Step>
  <Step title="Authenticate">
    Include your API key in every request using the `x-api-key` header.
    ```bash
    curl -H "x-api-key: YOUR_API_KEY" \
      "https://01resolved.var-meta.com/api/v1/dao/treasury/overview?slug=metadao"
    ```
  </Step>
  <Step title="Make your first call">
    Get a live snapshot of every tracked project: token price, treasury, NAV, mNAV, and runway: in a single call.
    ```bash
    curl -H "x-api-key: YOUR_API_KEY" \
      "https://01resolved.var-meta.com/api/v1/global-dashboard/projects-launch"
    ```
  </Step>
</Steps>

## Base URL

```
https://01resolved.var-meta.com
```

All endpoints are prefixed with `/api/v1/`.
