## GET /v1/global-dashboard/projects-launch

Returns all tracked projects with live token metrics, treasury data, NAV, and mNAV. **The most useful single endpoint for a weekly ecosystem snapshot.**

**Parameters:** `limit`, `page`

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://api.01resolved.com/v1/global-dashboard/projects-launch"
```

<Accordion title="Example Response (one project)">
```json
{
  "meta": { "totalItems": 12 },
  "data": [
    {
      "organizationSlug": "metadao",
      "organizationName": "MetaDAO",
      "tokenSymbol": "META",
      "tokenUsdPrice": "3.3260",
      "tokenPriceChangePercentage1h": "0.41",
      "tokenPriceChangePercentage24h": "-1.98",
      "tokenPriceChangePercentage7d": "9.46",
      "tokenCirculatingSupply": "22684697.85",
      "tokenTotalSupply": "22684697.85",
      "marketCap": "75432981.00",
      "fdv": "75432981.00",
      "treasuryValue": "12476348.74",
      "spendingLimit": "240000",
      "proposalCount": 37,
      "netAssetValue": "0.558",
      "monthsOfRunway": "44.67",
      "mNAV": "6.04",
      "premDisc": "497.31"
    }
  ]
}
```
</Accordion>

| Field | Description |
|-------|-------------|
| `tokenUsdPrice` | Current token price in USD |
| `tokenPriceChangePercentage1h` | Price change over the last hour (%) |
| `tokenPriceChangePercentage24h` | Price change over the last 24 hours (%) |
| `tokenPriceChangePercentage7d` | Price change over the last 7 days (%) |
| `tokenCirculatingSupply` | Tokens publicly available and actively circulating |
| `tokenTotalSupply` | Total tokens in existence (vested and unvested). Formula: `total minted - total burned` |
| `marketCap` | Market capitalization. Formula: `token price × circulating supply` |
| `fdv` | Fully diluted valuation. Formula: `token price × total supply` |
| `treasuryValue` | Total value of non-native assets held by the protocol treasury |
| `spendingLimit` | Monthly spend limit in USDC |
| `netAssetValue` | NAV (AS): non-native asset value divided by adjusted supply. Per-token backing in USD. |
| `mNAV` | Fully diluted market cap divided by total NAV. Formula: `FDV / NAV`. High = trading at a premium to non-native asset backing. |
| `premDisc` | Spot premium or discount to NAV (AS). Formula: `(spot price - NAV(AS)) / NAV(AS) × 100`. Positive = premium. |
| `monthsOfRunway` | Months of runway. Formula: `treasury value / (monthly spend limit + other monthly expenses)` |

## GET /v1/global-dashboard/decision-markets

Returns all proposals across all DAOs. **The best endpoint for monitoring ecosystem-wide governance.**

**Parameters:** `limit`, `page`

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://api.01resolved.com/v1/global-dashboard/decision-markets?limit=50"
```

**Response per proposal:** `project`, `projectSlug`, `proposal`, `publicKey`, `status`, `result`, `startDate`, `endDate`, `size`, `volume`, `trades`, `uniqueTraders`

Filter on `status` for `live` proposals. Check `volume` to prioritize by significance.

## GET /v1/global-dashboard/projects

Returns all tracked DAOs with proposal activity summary.

**Parameters:** `limit`, `page`

**Response per DAO:** `organizationSlug`, `organizationName`, `proposalCount`, `lastProposalStatus`, `lastProposalEndDate`

## GET /v1/global-dashboard/projects/decision-markets

Returns currently live or initiated proposals with staking progress and TWAP status.

| Field | Description |
|-------|-------------|
| `stakingProgressPercentage` | % of required stake accumulated |
| `proposalTwapStatus` | Current market signal: `approve`, `reject`, or `pending` |

## GET /v1/global-dashboard/completed-launches/info

Returns completed ICOs with full fundraising data and current ICO return.

**Parameters:** `limit`, `page`

<Accordion title="Example Response (one ICO)">
```json
{
  "data": [
    {
      "title": "MetaDAO",
      "tokenSymbol": "META",
      "tokenUsdPrice": "3.3260",
      "icoPrice": "0.8054",
      "icoReturn": "313.18",
      "fundRaised": "8054207",
      "fundsAccepted": "8054207",
      "icoInvestors": 5035,
      "holdersCount": 5099,
      "organizationSlug": "metadao"
    }
  ]
}
```
</Accordion>

| Field | Description |
|-------|-------------|
| `icoReturn` | % return from ICO price to current price. Formula: `((current price - ICO price) / ICO price) × 100` |
| `fundRaised` | Total capital committed during the ICO |
| `fundsAccepted` | Capital actually accepted (may be less if oversubscribed) |

## GET /v1/global-dashboard/recent-launches

Returns recent ICO launches including live and upcoming.

**Parameters:** `limit`, `page`

**Response per launch:** `launchAddr`, `title`, `tokenSymbol`, `state`, `minimumRaiseAmount`, `committedAmount`, `organizationSlug`

---

## GET /v1/global-dashboard/graveyard-launches

Returns failed or liquidated launches: projects that did not meet their minimum raise.

**Response:** `{title, tokenSymbol, reasonStatus: "liquidated", minimumRaiseAmount, committedAmount, fundsAccepted, icoInvestors}`

**Ranger example:** $86.4M committed, $8M accepted, 8,761 investors: liquidated.
