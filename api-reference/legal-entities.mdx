## GET /api/v1/legal-entity

Returns legal entity registrations for tracked projects.

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://api.01resolved.com/api/v1/legal-entity"
```

<Accordion title="Example Response">
```json
{
  "data": [
    {
      "project": "avici",
      "type": "DAO",
      "legalName": "Jester DAO LLC",
      "entityName": "Avici DAO",
      "jurisdiction": "Marshall Islands"
    },
    {
      "project": "avici",
      "type": "OpCo",
      "legalName": "Avici Inc.",
      "entityName": "Avici OpCo.",
      "jurisdiction": "Delaware"
    }
  ]
}
```
</Accordion>

| Field | Description |
|-------|-------------|
| `type` | Entity type: `DAO`, `OpCo`, etc. |
| `legalName` | Registered legal name of the entity |
| `entityName` | Operational name used by the project |
| `jurisdiction` | Legal jurisdiction (e.g. `Marshall Islands`, `Delaware`) |
