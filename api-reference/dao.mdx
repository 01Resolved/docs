## GET /api/v1/dao/overview

Returns on-chain configuration and token metadata for a DAO.

**Parameters:** `slug` (query, string)

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/dao/overview?slug=metadao"
```

<Accordion title="Example Response">
```json
{
  "data": {
    "name": "MetaDAO",
    "slug": "metadao",
    "publicKey": "CUPoiqkK4hxyCiJcLC4yE9AtJP1MoV1vFV2vx3jqwWeS",
    "quoteMint": "EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v",
    "baseMint": "METAwkXcqyXKy1AtsSgJ8JiUHwGCafnZL38n3vYmeta",
    "spendingLimit": "240000000000",
    "passedThresholdBps": "300",
    "baseToStake": "200000000000",
    "twapStartDelaySeconds": "86400",
    "version": "0.6",
    "socials": [
      { "type": "x", "value": "https://x.com/MetaDAOProject" }
    ],
    "baseToken": {
      "symbol": "META",
      "decimals": 6,
      "usdPrice": "3.34",
      "totalSupply": "22684697.85",
      "holders": "5099"
    }
  }
}
```
</Accordion>

| Field | Description |
|-------|-------------|
| `spendingLimit` | Max per-proposal spend in raw token units. Divide by 10^6 for USDC value |
| `passedThresholdBps` | TWAP pass threshold in basis points (300 = 3%) |
| `baseToStake` | Governance tokens required to open a proposal. Divide by 10^6 |
| `twapStartDelaySeconds` | Seconds after market open before TWAP starts (86400 = 24h) |

## GET /api/v1/dao/treasury/overview

Returns high-level treasury summary. The fastest way to get key financial metrics.

**Parameters:** `slug` (query, string)

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/dao/treasury/overview?slug=metadao"
```

<Accordion title="Example Response">
```json
{
  "data": {
    "baseMintCurrentPrice": "3.3374536307464315",
    "totalBalance": "12476348.74",
    "netAssetValue": "0.5579506169974838",
    "spendingLimit": "240000",
    "monthOfRunway": "44.67"
  }
}
```
</Accordion>

| Field | Description |
|-------|-------------|
| `totalBalance` | Total treasury value in USD |
| `netAssetValue` | NAV — treasury value divided by circulating supply (per-token USD book value) |
| `spendingLimit` | Monthly spending allowance in USDC |
| `monthOfRunway` | Months of runway. Formula: `treasury value / (monthly spend limit + other monthly expenses)` |

## GET /api/v1/dao/treasury

Returns the full treasury breakdown — every tracked wallet and every token held.

**Parameters:** `slug` (query, string)

| Field | Description |
|-------|-------------|
| `totalAssetsValue` | Total USD value across all accounts |
| `groupedAssets[].treasuryAddress` | The wallet holding this token |
| `groupedAssets[].amount` | Raw token amount. Divide by `10^decimals` for human-readable value |

<Note>Treasury value includes assets held in DAO wallets **and** LP positions.</Note>

## GET /api/v1/dao/account-registry

Returns all wallets tracked for a DAO with balances, names, and custody type.

**Parameters:** `slug`

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/dao/account-registry?slug=metadao"
```

| Field | Description |
|-------|-------------|
| `name` | Labeled wallet name (e.g. `Main`, `Market Making`, `FutarchyAMM`) |
| `custodyType` | How the wallet is controlled (e.g. `Multi-sig`) |
| `isActive` | Whether the wallet is currently in use |

---

## GET /api/v1/dao/movements

Returns classified treasury transactions with counterparty information.

**Parameters:** `slug`, `limit`, `page`

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://01resolved.var-meta.com/api/v1/dao/movements?slug=metadao&limit=10"
```

| Field | Description |
|-------|-------------|
| `signature` | Solana transaction hash |
| `value` | USD value at time of transaction |
| `treasury.name` | Labeled name of the sending wallet |
| `relatedTreasury.name` | Labeled name of the receiving wallet (if a known DAO wallet) |

## GET /api/v1/dao/token-supply

Returns all token mint and burn events linked to governance proposals.

**Parameters:** `slug`

| Field | Description |
|-------|-------------|
| `type` | `Mint` or `Burn` |
| `amount` | Token amount |
| `resultingSupply` | Supply after this event |
| `priceAtTime` | Token price at event time |
| `proposal` | Linked governance proposal |

---
