## GET /v1/trading-activity/{daoSlug}

Returns aggregate decision market trading statistics for a DAO.

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://api.01resolved.com/v1/trading-activity/metadao"
```

**Response:** `{totalVolume, totalTrades, totalUniqueTraders}`

**MetaDAO example:** $6.4M total volume, 134,850 trades, 499 unique traders.

---

## GET /v1/trading-activity/{daoSlug}/trading-activities

Returns all individual trades across all proposals for a DAO, most recent first.

**Parameters:** `daoSlug` (path), `limit`, `page`

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://api.01resolved.com/v1/trading-activity/metadao/trading-activities?limit=10"
```

## GET /v1/trading-activity/special-wallet

Returns all decision market trades placed by a specific wallet address.

**Parameters:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `walletPublicKey` | string | **Yes** | Solana wallet address |
| `slug` | string | No | Filter to a specific project |
| `limit` | number | No | Items per page |
| `page` | number | No | Page number |
| `sort` | string | No | Sort field: `timestamp`, `value`, `price`, `size`, `proposalVolumeConc`, `marketDirectionConc`, `proposalTradeConc` |
| `order` | string | No | `ASC` or `DESC` (default `DESC`) |

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://api.01resolved.com/v1/trading-activity/special-wallet?walletPublicKey=AWjCy...SC5&limit=10"
```

<Warning>
  The parameter name is `walletPublicKey`, not `walletAddress` or `wallet`. Incorrect names return a 400 error.
</Warning>

## GET /v1/trading-activity/special-wallet/overview

Returns aggregate trading statistics for a specific wallet address.

**Parameters:** `walletPublicKey` (**required**), `slug` (optional)

```bash
curl -H "x-api-key: YOUR_KEY" \
  "https://api.01resolved.com/v1/trading-activity/special-wallet/overview?walletPublicKey=AWjCy...SC5"
```

| Field | Description |
|-------|-------------|
| `totalVolume` | Total USD traded by this wallet |
| `totalTrades` | Total trades placed |
| `totalUniqueProposals` | Number of distinct proposals traded. Signals whether the wallet is a broad participant or focused. |
| `walletCreationTime` | Unix timestamp of wallet creation. Use to distinguish long-term vs new wallets. |
