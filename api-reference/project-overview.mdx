All project overview endpoints accept a `slug` query parameter.

---

## GET /api/v1/project-overview

Returns token metrics and project description.

**Parameters:** `slug`

<Accordion title="Example Response">
```json
{
  "data": {
    "baseMint": "METAwkXcqyXKy1AtsSgJ8JiUHwGCafnZL38n3vYmeta",
    "tokenSymbol": "META",
    "tokenPrice": "3.3260",
    "tokenPriceUSDChange24h": "-1.98",
    "volumeChanged24h": "13.42",
    "totalSupply": "22684697.85",
    "circulatingSupply": "22684697.85",
    "icoPrice": 0.8054207616,
    "projectDescription": "MetaDAO escapes politics by using futarchy."
  }
}
```
</Accordion>

| Field | Description |
|-------|-------------|
| `tokenPriceUSDChange24h` | 24h price change %. Formula: `((price_now - price_24h_ago) / price_24h_ago) × 100` |
| `volumeChanged24h` | 24h volume **change** percentage (not raw volume). Formula: `((volume_24h - volume_prev_24h) / volume_prev_24h) × 100` |
| `totalSupply` | Total tokens in existence. Formula: `total minted - total burned` |
| `circulatingSupply` | Tokens in circulation excluding locked team allocations. Formula: `total supply - performance package tokens` |
| `icoPrice` | Token price at time of ICO |

## GET /api/v1/project-overview/ico-details

Returns ICO-specific details including raise timing and token allocation.

**Parameters:** `slug`

| Field | Description |
|-------|-------------|
| `icoPrice` | Token price during the ICO |
| `unixTimestampStarted` | Unix timestamp when the ICO started |
| `secondsForLaunch` | Duration of the ICO in seconds |
| `icoSupply` | Tokens allocated to the ICO |

---

## GET /api/v1/project-overview/ico-contributor

Returns aggregate ICO contributor statistics.

**Parameters:** `slug`

| Field | Description |
|-------|-------------|
| `totalContributors` | Unique wallets that contributed |
| `totalTransactions` | Total contribution transactions |
| `totalQuoteAmount` | Total USDC contributed |
| `averageContribution` | Average contribution per wallet in USDC |
| `medianContribution` | Median contribution per wallet in USDC |
| `fundsAccepted` | Total funds accepted by the project |

---

## GET /api/v1/project-overview/ico-contributor/investors

Returns per-wallet ICO contribution data.

**Parameters:** `slug`, `limit`, `page`

---

## GET /api/v1/project-overview/token-distribution

Returns token allocation breakdown by category.

**Parameters:** `slug`

**MetaDAO example:** 98.5% Public Sale / ICO, 1.5% Liquidity.

**Response:** `{totalSupply, categories: [{allocation, amount, percentage}]}`

---

## GET /api/v1/project-overview/legals

Returns legal document links for a project.

**Parameters:** `slug`

---

## GET /api/v1/project-overview/performance-package

Returns vesting and team token allocation data.

**Parameters:** `slug`

| Field | Description |
|-------|-------------|
| `totalInPackage` | Total tokens in team vesting package |
| `locked` | Tokens not yet vested |
| `claimableNow` | Tokens vested but not yet claimed |
| `claimed` | Tokens already claimed |
| `cliff` | Vesting cliff date |

---

## GET /api/v1/project-overview/tranche-ladder

Returns individual vesting tranches.

**Parameters:** `slug`

---

## GET /api/v1/project-overview/unlock-schedule

Returns the token unlock schedule chart data.

**Parameters:** `slug`
